Software elaborado como desafío para a seleção de estágio da Superintendência de Tecnologia da Informação (STI) da Secretária de
Estado do Planejamento, Gestão e Patrimônio (SEPLAG).

Neste desafío, tem como objetivo analisar:

	- Orientação a Objetos;
	- Lógica de Programação;
	- Banco de Dados.

A linguagem de programação escolhida para o desenvolvimento do sistema foi o Python, junto com seu framework web, o Django. Tendo
como Banco de Dados escolhido o PostgreSQL.

O Software:

- O Software consiste em um sistema de pesca que permite localizar o pescador, registrar seus resultados de cada pescaria e ranquear
cada pescador em dois rank: o rank de massa, onde contempla os 3 pescadores que pescaram o peixe com maior massa(kg) e o rank de 
quantidade, que contempla os 3 pescadores que pescaram a maior quantidade de peixes.
